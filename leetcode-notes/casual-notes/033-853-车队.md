## 解题思路
注意，/ 是真除法，结果不是取整，但是python2的话是取整

## 代码
```python
class Solution(object):
    def carFleet(self, target, position, speed):
        """
        :type target: int
        :type position: List[int]
        :type speed: List[int]
        :rtype: int
        """
        cars = sorted(zip(position, speed), key = lambda x: -x[0])
        max_time = 0
        res = 0
        for pos, sp in cars:
            time = (target - pos) / sp
            if time > max_time:
                max_time = time
                res += 1
        return res
```
